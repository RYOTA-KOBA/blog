# ブログをはじめました

このたび、Next.js と Markdown を使用した個人ブログを立ち上げました。

## ブログの特徴

このブログは以下の特徴を持つ、シンプルで拡張可能なブログシステムです：

### 1. **Next.js による高速な動作**
React ベースの最新フレームワークで、SSR/SSG に対応しています。

### 2. **Markdown で執筆**
記事は Markdown 形式で執筆します。Front Matter でメタデータを管理できます。

### 3. **自動分類機能**
- カテゴリ別の記事一覧
- 月別のアーカイブ
- これらはサイドバーから簡単にアクセス可能

### 4. **Cloudflare Pages でホスティング**
Terraform で完全に管理されたインフラストラクチャです。

## 記事投稿方法

記事を投稿するには、`public/articles/` ディレクトリに新しい Markdown ファイルを作成します。

詳細は README.md をご参照ください。
